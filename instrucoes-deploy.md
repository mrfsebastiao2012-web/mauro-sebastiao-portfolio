# 🚀 Portfolio de Mauro Sebastião

## Deploy Gratuito em 3 Passos

### Opção 1: Netlify (Recomendado - mais rápido)

1. Aceda a [netlify.com](https://netlify.com) e registe-se gratuitamente
2. Na dashboard, arraste e largue a pasta `portfolio/` para a zona de deploy
3. O seu site fica online em segundos com URL tipo: `https://mauro-sebastiao.netlify.app`
4. **Opcional**: Configure domínio personalizado (ex: `maurosebasitao.netlify.app`)

### Opção 2: Vercel

1. Aceda a [vercel.com](https://vercel.com) e registe-se
2. Clique em "Add New" → "Project"  
3. Arraste a pasta ou conecte ao GitHub
4. URL tipo: `https://mauro-sebastiao.vercel.app`

### Opção 3: GitHub Pages (gratuito)

1. Crie conta em [github.com](https://github.com)
2. Crie repositório público com nome `mauro-sebastiao.github.io`
3. Faça upload do `index.html`
4. Em Settings → Pages → Source: `main branch`
5. URL: `https://mauro-sebastiao.github.io`

## ⚡ O que pode gerir no Admin:

| Secção | O que pode fazer |
|--------|------------------|
| **Perfil** | Nome, título, bio, foto, contactos |
| **Experiência** | Adicionar/remover posições profissionais |
| **Competências** | Gerir skills técnicas com barras de progresso |
| **Galeria** | Upload de fotos e vídeos dos seus projectos |
| **Certificações** | Adicionar/remover certificações |
| **Definições** | Alterar password, exportar/importar backup |

## 📧 Para personalizar com os seus dados reais:

Após o deploy, faça login como Admin e actualize:
1. **Perfil**: Adicione a sua foto, email real, telefone, localização
2. **Experiência**: Substitua pelos seus empregos reais (do LinkedIn)
3. **Competências**: Ajuste as tecnologias que domina
4. **Certificações**: Adicione as suas certificações reais
5. **Galeria**: Carregue fotos dos seus projectos
